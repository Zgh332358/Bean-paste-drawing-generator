## 立即可执行的任务

### 任务A：在线调研（无需购买）

**目标：** 收集市场信息，验证色号配置

**步骤：**
1. 访问淘宝/拼多多搜索"拼豆套装 48色"
2. 筛选销量前10的产品
3. 查看产品详情页的色卡图片
4. 记录以下信息：
   - 店铺名称和产品链接
   - 色号编码方式（01-48 或其他）
   - 色卡图片（截图保存）
   - 用户评价中关于颜色的反馈

**输出：**
- 创建 `market-research-findings.md` 文档
- 保存色卡图片到 `docs/color-cards/` 目录
- 更新色号映射表（如有差异）

### 任务B：图像色彩提取（技术方案）

**目标：** 从色卡图片提取RGB值

**工具：**
- Photoshop / GIMP（取色器）
- 在线工具：https://imagecolorpicker.com/
- Python脚本（PIL库）

**步骤：**
1. 打开色卡图片
2. 对每个颜色块取样（避开边缘）
3. 记录RGB值
4. 计算平均值（如有多个样本）

**输出：**
- 更新 `generic-palette-config.json`
- 记录取样方法和误差范围

### 任务C：用户验证（社区反馈）

**目标：** 从实际用户获取反馈

**渠道：**
- 小红书：搜索"拼豆"相关笔记
- B站：观看拼豆教程视频
- 贴吧/豆瓣：拼豆爱好者社区
- QQ群/微信群：拼豆交流群

**收集信息：**
- 用户常用的套装品牌
- 色号使用习惯
- 颜色匹配的痛点

**输出：**
- 用户反馈摘要
- 常见问题列表
- 优化建议

## 需要购买实物的任务（可选）

**用途：**

### 任务E：专业色彩校准（高级）

**工具需求：**
- 色彩校准卡（X-Rite ColorChecker）
- 标准光源（D65）
- 分光光度计（可选）

**步骤：**
1. 在标准光源下拍摄拼豆
2. 使用校准卡进行白平衡
3. 测量每个颜色的Lab值
4. 更新色卡数据库
